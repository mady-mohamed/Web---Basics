Latest Projects

Pacman Game

This project is an implementation of the classic Pacman game built with Python. It features core game mechanics, including Pacman and ghost movement, score tracking, and level progression. A key aspect of this project is the integration of a simple A*, and greedy algorithms for intelligent ghost movement, with certain performance-critical elements of the A* algorithm implemented in C++ and integrated via `ctypes`. The game boasts a user- interface developed using Pygame and is structured into modular components for clear organization of game logic, settings, and menu handling. While currently Windows-only, the project demonstrates a solid understanding of game development principles and algorithm implementation.
